# Explainable ML Engine

## Overview

This project focuses on building an explainable machine learning system that not only predicts outcomes but also explains the reasoning behind predictions using SHAP and LIME.

## Features

* Machine Learning Prediction API
* Model Explainability (SHAP, LIME)
* Comparison of explanation techniques
* Scalable backend with FastAPI

## Tech Stack

* FastAPI
* Scikit-learn
* SHAP
* LIME
* MongoDB (planned)
